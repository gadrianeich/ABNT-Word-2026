# ABNT Word 2026 📋

Template completo de normas ABNT para Microsoft Word 2016 e superiores.

## 🎯 O que é este projeto?

Este repositório contém estilos e configurações prontas para formatar documentos acadêmicos e profissionais de acordo com as **Normas Brasileiras de Associação Técnica (ABNT)**.

## 📦 Arquivos Inclusos

- `styles.xml` - Estilos ABNT em formato XML
- `GUIA_FORMATACAO_ABNT.md` - Guia completo de formatação
- `EXEMPLO_USO.md` - Exemplos práticos
- `template-abnt.docx` - Template Word pronto para usar

## 🚀 Como Usar

### Opção 1: Usar o Template Direto
1. Baixe o arquivo `template-abnt.docx`
2. Abra no Word
3. Comece a escrever seu documento

### Opção 2: Importar Estilos em um Documento Existente
1. Abra seu documento Word
2. Acesse: **Design** → **Estilos** → **Gerenciar Estilos** (ou pressione **Alt + Ctrl + Shift + S**)
3. Clique no ícone de menu (três pontos)
4. Selecione **Importar/Exportar**
5. Escolha o arquivo `styles.xml`

## 📐 Formatação ABNT Padrão

### Margens
- Superior: 3 cm
- Inferior: 2 cm
- Esquerda: 3 cm
- Direita: 2 cm

### Fonte
- Tipo: Times New Roman ou Arial
- Tamanho: 12 pt
- Espaçamento: 1,5 linha

### Títulos
- Título 1: 16 pt, Negrito, Maiúsculo
- Título 2: 14 pt, Negrito
- Título 3: 12 pt, Negrito, Itálico

## 📖 Elementos ABNT Inclusos

✅ Estilos de títulos e subtítulos
✅ Espaçamento de parágrafos correto
✅ Formatação de citações
✅ Estilos de referências
✅ Capa formatada ABNT
✅ Sumário automático

## 🔧 Requisitos

- Microsoft Word 2016 ou superior
- Windows ou Mac
- .NET Framework (Windows)

## 📚 Referências

- NBR 6023:2018 - Referências bibliográficas
- NBR 10520:2023 - Citações em documentos
- NBR 14724:2011 - Trabalhos acadêmicos

## 💡 Dicas

1. Use sempre os estilos predefinidos
2. Não formate manualmente (deixe os estilos fazerem isso)
3. Use o sumário automático do Word
4. Mantenha a consistência visual
